## Setup & Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/havenyj/EiThinzarMyo_DataPortfolio.git
   cd EiThinzarMyo_DataPortfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and go to `http://localhost:5173` to preview the portfolio.

---

## Deployment

**Build for production:**
```bash
npm run build
```

Deploy the contents of the `dist/` folder.  
If using **GitHub Pages**, configure the Pages source to the deploy branch (e.g., `gh-pages` or `main`) and ensure you're serving from the correct folder.
